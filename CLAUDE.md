This project is a repository for a university course in Ukrainian language.
The course is "Machine learning and artificial inelligence in game development".
The course gives foundational knowledge on machine learning, only going into specifics of
applications to game development in later lectures.

## 1. Rules that apply for all tasks:

1. Lectures and labs are generated in Ukrainian.
2. Load context automatically from rules, skills, memory files.
3. Do not load project files into context automatically, unless mentioned in conversation or directed to do so

## 2. Course plan.

2.1. Lectures plan: ./memory/lectures-plan.md
2.2. Labs plan: ./memory/labs-plan.md

## 3. Types of generated content:

3.1. lecture notes: memory file is ./memory/lectures-sources.md
3.2. lab notes: memory file is ./memory/
